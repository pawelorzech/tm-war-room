A player log analysis tool. All data stored in browser, no server-side storage.

**Features:**

- Performance tracking: crime/OC count, success streak, failure rate
- Date range filtering, interactive charts, CSV/JSON export
- Mug Summary table with expandable rows
- Requires Full Access API key

### 2. FFScouter (v3) — Free Stat Predictor & Target List

**Author:** Glasnost [1844049]
**Rating:** 207 | **Replies:** 131
**URL:** https://ffscouter.com
**GitHub:** https://github.com/rDacted2/fair_fight_scouter

**Users:** ~28,768 registered API keys, ~20,975 active, 470,733+ total stat estimates

**Features:**

1. **Stat Predictor (core)** — Estimates opponent battle stats from crowdsourced Fair Fight scores. Most accurate free tool on Torn. Less accurate above 25bn total stats.

2. **Target Finder** — Preset modes: Levelling List, Respect/Chain List, Custom (filter by level 1-100, FF range, inactive, factionless), Manual, Not in Hosp List. Live status updates every 60s with Hospital/Jail countdown timers. Bookmark filter configs.

3. **War Room** — Real-time enemy faction monitoring. Displays level, status, FF value, battle stat estimate, last action. Configurable refresh (10-60s). Quick-access attack buttons. History panel tracking status changes. Filter by online/offline, sort by activity or FF.

4. **Player Viewer** — Look up any player by ID.

5. **FF Scouter Userscript** (companion by Weav3r) — FF estimates on profiles, FF columns in faction lists, color-coded attack history, honor bar indicators.

6. **Public API** — Used by TornTools, TornPDA, and other tools.

**API usage:** Up to 13 requests/day per user. Requires basic, battlestats, attacks, hof, personalstats permissions.

### 3. Torn Stock Advisor

**Author:** Nicram [3702492]
**Rating:** 21 | **Replies:** 27
**Format:** Tampermonkey userscript + TornPDA

**Features:**

**Tab 1 — Signals:**
- BUY: High ROI benefit blocks (PTS, SYM, HRG, MUN, FHG) ranked by annual return + elite passives (TCB, WSU, WLT). Cash-payout BBs compared against Investment Bank APR; suppressed if bank pays more.
- SWING: Bullish momentum (RSI oversold + uptrend) on unheld stocks
- TRIM: In profit with shares above BB minimum
- TAKE PROFIT: Overbought + in profit + no BB
- SELL: Bearish momentum + losing position + no BB
- COLLECT: Unclaimed dividend ready
- HOLD: BB active, keep. Warns if bank APR beats BB's return.

**Tab 2 — Portfolio:** All positions with P/L %, dividend progress, BB status, total value.

**Tab 3 — Market Scan:** Every stock ranked by momentum score, signal direction, RSI, 7-day change, BB cost/ROI. Flags where bank APR > BB.

**Technical details:**
- Benefit ROI engine: annual return per BB; only >5% annual or top-tier passives get BUY
- Bank vs BB comparison: fetches live APRs for all 5 terms (1w/2w/1m/2m/3m)
- Technical analysis via Tornsy.com: SMA(7), SMA(20), RSI(14), 7d/30d price change, 30d range position, composite momentum
- Local price tracking: snapshots every 60s in storage
- API: 3 calls per 60s refresh, read-only, Limited Access key sufficient
- Open source (MIT), privacy-safe

**Data sources:** Torn API v1 (stocks, user stocks, bank rates) + Tornsy.com (OHLC candles, free, no key)
